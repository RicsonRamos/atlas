## Conteúdo programático

**Fundamentos** — A dimensão temporal na análise espacial (o crime não ocorre aleatoriamente no espaço nem no tempo); conceitos de sazonalidade criminal (efeito de clima, feriados, horários comerciais); deslocamento espacial do crime (tático, temporal, geográfico, de alvo) decorrente de intervenções policiais.

**Teoria** — Modelagem espaço-temporal por métodos estatísticos autoregressivos; metodologia de Risk Terrain Modeling (RTM - conceituação de Caplan e Kennedy), que avalia como as características físicas do ambiente (bares, becos escuros, saídas de metrô) atuam cooperativamente como geradores ou atratores de crimes; introdução a algoritmos de Machine Learning (Florestas Aleatórias, Redes Neurais Espaciais) aplicados à predição criminal em grades regulares (grid cell forecasting).

**Aplicação prática** — Implementação de um RTM simplificado em Python para calcular o escore de risco de células urbanas com base na proximidade de dois fatores de risco ambientais.

## Código de Exemplo em Python (Risk Terrain Modeling - RTM Simplificado)
```python
import numpy as np
import pandas as pd

# Definir tamanho da grade urbana (5x5 blocos)
tamanho_grade = 5

# Localização de dois fatores de risco (coordenadas x, y na grade)
fatores_risco = {
    'bares': [(1, 1), (3, 2)],
    'becos_escuros': [(3, 4), (0, 2)]
}

# Inicializar matriz de risco com zeros
grade_risco = np.zeros((tamanho_grade, tamanho_grade))

# Função para aplicar escore de risco baseado em distância de Manhattan
def aplicar_rtm(grade, fatores, peso_bar=2, peso_beco=3):
    for x in range(tamanho_grade):
        for y in range(tamanho_grade):
            # Calcular proximidade aos bares (risco decresce com a distância)
            for bx, by in fatores['bares']:
                dist = abs(x - bx) + abs(y - by)
                if dist <= 1:
                    grade[x, y] += peso_bar
            # Calcular proximidade a becos escuros
            for ex, ey in fatores['becos_escuros']:
                dist = abs(x - ex) + abs(y - ey)
                if dist <= 1:
                    grade[x, y] += peso_beco
    return grade

escores_finais = aplicar_rtm(grade_risco, fatores_risco)
df_grade = pd.DataFrame(escores_finais)
print("Matriz de Risco do Terreno (Escores RTM):")
print(df_grade)
print("\nCélula com maior atratividade de risco criminal:", np.unravel_index(escores_finais.argmax(), escores_finais.shape))
```

## Estado da arte e debates em aberto
- **Consenso técnico:** Células espaciais que sofrem incidentes de roubo possuem alta probabilidade de sofrer novos roubos no curto prazo (Near-Repeat), tornando modelos autoregressivos úteis para previsões semanais.
- **Debate ativo:** O viés de feedback em algoritmos de Machine Learning de predição: o envio de patrulhas com base em previsões automatizadas gera maior quantidade de registros e prisões nessas mesmas áreas, retroalimentando o modelo artificialmente e criando hotspots permanentes de policiamento injustificado.
