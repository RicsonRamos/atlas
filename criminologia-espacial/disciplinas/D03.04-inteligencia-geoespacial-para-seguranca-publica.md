## Código de Exemplo em Python (Integração de Dados Policiais e Censitários)
```python
import pandas as pd
import geopandas as gpd
from shapely.geometry import Point

# 1. Simulação de polígonos de bairros com dados populacionais do Censo
bairros_dados = {
    'bairro': ['Centro', 'Bairro_A', 'Bairro_B'],
    'populacao': [10000, 25000, 15000],
    'geometry': [
        Point(0, 0).buffer(2),  # Polígono simplificado
        Point(4, 0).buffer(2),
        Point(0, 4).buffer(2)
    ]
}
gdf_bairros = gpd.GeoDataFrame(bairros_dados, crs="EPSG:3857")

# 2. Simulação de coordenadas de roubos (pontos)
ocorrencias_dados = {
    'id_crime': [1, 2, 3, 4, 5],
    'geometry': [Point(0.5, 0.5), Point(0.1, -0.2), Point(3.8, 0.2), Point(4.1, -0.1), Point(0.2, 3.9)]
}
gdf_crimes = gpd.GeoDataFrame(ocorrencias_dados, crs="EPSG:3857")

# 3. Junção espacial (Spatial Join): verificar em qual bairro cada crime ocorreu
gdf_join = gpd.sjoin(gdf_crimes, gdf_bairros, how="inner", predicate="within")

# 4. Calcular taxa de crimes por 100.000 habitantes
contagem_crimes = gdf_join.groupby('bairro').size().rename('total_crimes')
gdf_consolidado = gdf_bairros.set_index('bairro').join(contagem_crimes).fillna(0)
gdf_consolidado['taxa_por_100k'] = (gdf_consolidado['total_crimes'] / gdf_consolidado['populacao']) * 100000

print("Relatório de Inteligência Geoespacial Consolidade por Bairro:")
print(gdf_consolidado[['populacao', 'total_crimes', 'taxa_por_100k']])
```
