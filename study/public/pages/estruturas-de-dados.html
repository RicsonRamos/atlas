Estruturas de Dados

A escolha da estrutura de dados correta pode ser a diferença entre um sistema que processa milhões de requisições por segundo e um que falha sob carga.

Além do Básico

Entenda interativamente como funcionam as engrenagens dos sistemas de software modernos.

B+ Trees

Projetadas para sistemas de disco (bancos de dados relacionais). Minimizam as operações de I/O por terem alto fator de ramificação.

Hash Tables

O(1) para buscas. A base de caches (Redis) e dicionários de linguagens. Utilizam funções hash para mapear chaves a índices.

Bloom Filters

Estrutura probabilística. Troca 100% de precisão (falsos positivos possíveis) por altíssima eficiência de memória e velocidade.

Heaps

Fila de prioridade O(1) para o máximo/mínimo. Usados em agendamento do SO e roteamento de redes.

B+ Trees (Árvores B+)

B+ Trees mantêm os dados ordenados e permitem buscas eficientes com poucos acessos a disco (I/O) devido ao seu alto fator de ramificação ("fanout").

Casos de Uso Comuns

• Índices de BD (MySQL, Postgres)
• Sistemas de Arquivos (ext4, NTFS)
• Range Queries (Consultas por intervalo)

Hash Tables (Encadeamento)

Mapeiam chaves para valores computando um índice através de uma Função Hash. Explore como colisões são tratadas usando encadeamento (listas ligadas nos buckets).

Casos de Uso Comuns

• Caches (Memcached, Redis)
• Dicionários (Python dicts, JS Objects)
• Tabelas de Roteamento

Neste simulador, a função Hash calcula o (Soma dos caracteres) % 5.

Bloom Filters

Estrutura de dados probabilística usada para testar se um elemento é membro de um conjunto. Falsos positivos são possíveis, mas falsos negativos não.

Casos de Uso Comuns

• Prevenir I/O de disco desnecessário (Cassandra)
• Filtros de URLs maliciosas no Chrome
• Recomendadores de conteúdo (não sugerir itens já vistos)

Heaps (Filas de Prioridade)

Garantem acesso O(1) constante ao elemento "máximo" ou "mínimo", mantendo a árvore parcialmente ordenada a cada inserção/extração (bubble up / sink down).

Casos de Uso Comuns

• Roteamento (Dijkstra)
• Agendamento do SO
• Ordenação HeapSort
