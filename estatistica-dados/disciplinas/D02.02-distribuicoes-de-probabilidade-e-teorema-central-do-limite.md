## Conteúdo Programático
**Fundamentos**: Parâmetros de distribuição vs. Estatísticas amostrais; a importância da modelagem paramétrica.
**Teoria Estatística**: Distribuição Binomial (ensaios independentes de Bernoulli); distribuição de Poisson (eventos raros em intervalo contínuo); distribuição Normal (simetria e a regra empírica 68-95-99.7); distribuição t de Student (caudas pesadas para amostras pequenas, graus de liberdade); a formulação do Teorema Central do Limite: dado um conjunto de variáveis aleatórias independentes e identicamente distribuídas (i.i.d.) com média $\mu$ e desvio padrão $\sigma$, a média amostral $\bar{X}$ converge em distribuição para $N(\mu, \sigma/\sqrt{n})$ quando $n \to \infty$.
**Implementação em Python**: Modelagem paramétrica com `scipy.stats`; geração de números pseudoaleatórios (`.rvs()`); simulação de Monte Carlo para demonstração visual do TCL.

## Implementação em Python
```python
import numpy as np
import pandas as pd
from scipy import stats

# 1. Modelar um problema com Poisson e Binomial
# Poisson: Uma API recebe em média 3 chamadas por minuto. Probabilidade de receber exatamente 5 chamadas em 1 min?
prob_poisson = stats.poisson.pmf(5, mu=3)
print(f"P(X=5 API calls | λ=3): {prob_poisson:.4f}")

# Binomial: Taxa de conversão de anúncios é 2%. Probabilidade de conseguir 4 ou mais conversões em 100 cliques?
# P(X >= 4) = 1 - P(X <= 3)
prob_binomial = 1 - stats.binom.cdf(3, n=100, p=0.02)
print(f"P(X >= 4 conversões | n=100, p=0.02): {prob_binomial:.4f}")


# 2. Simulação do Teorema Central do Limite (TCL)
# População original: Distribuição Exponencial (fortemente assimétrica, não-normal)
tamanho_populacao = 100000
media_pop = 10
populacao = np.random.exponential(scale=media_pop, size=tamanho_populacao)

# Coletar amostras e calcular suas médias
tamanho_amostra = 30 # N > 30 é o limite empírico clássico
numero_simulacoes = 1000
medias_amostrais = []

for _ in range(numero_simulacoes):
    amostra = np.random.choice(populacao, size=tamanho_amostra)
    medias_amostrais.append(amostra.mean())

# Estatísticas das médias amostrais
media_das_medias = np.mean(medias_amostrais)
desvio_padrao_das_medias = np.std(medias_amostrais)
erro_padrao_teorico = media_pop / np.sqrt(tamanho_amostra)

print("\n--- Resultados da Simulação do TCL ---")
print(f"Média Real da População: {media_pop}")
print(f"Média das Médias Amostrais: {media_das_medias:.4f} (Deve ser ~10)")
print(f"Desvio Padrão Real das Médias: {desvio_padrao_das_medias:.4f}")
print(f"Erro Padrão Teórico (σ / √n): {erro_padrao_teorico:.4f} (Deve ser ~1.82)")
```
