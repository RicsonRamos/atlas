## Conteúdo Programático
**Fundamentos**: Amostragem probabilística vs. Não-probabilística; viés de seleção; a diferença entre parâmetro (população) e estatística/estimador (amostra).

## Implementação em Python
```python
import numpy as np
import pandas as pd
from scipy import stats

# Criar um conjunto de dados não-normal (exemplo: tempo gasto por sessão em segundos)
np.random.seed(42)
tempos_sessao = np.random.lognormal(mean=2, sigma=0.8, size=150)

# 1. Método Clássico usando Distribuição t (Assumindo Normalidade das médias via TCL)
media_amostral = np.mean(tempos_sessao)
erro_padrao = stats.sem(tempos_sessao) # Desvio padrão amostral / sqrt(n)

ic_t = stats.t.interval(confidence=0.95, df=len(tempos_sessao)-1, 
                        loc=media_amostral, scale=erro_padrao)
print(f"Média Amostral: {media_amostral:.3f}")
print(f"IC 95% Clássico (t-Student): [{ic_t[0]:.3f}, {ic_t[1]:.3f}]")


# 2. Método Bootstrap (Mais robusto para métricas não normais e tamanho amostral limitado)
# Vamos estimar o Intervalo de Confiança de 95% da MEDIANA
n_bootstrap = 5000
medianas_boot = []

for _ in range(n_bootstrap):
    # Reamostrar com reposição
    reamostra = np.random.choice(tempos_sessao, size=len(tempos_sessao), replace=True)
    medianas_boot.append(np.median(reamostra))

# Método de Percentis
ic_mediana_inf = np.percentile(medianas_boot, 2.5)
ic_mediana_sup = np.percentile(medianas_boot, 97.5)

print(f"Mediana Amostral: {np.median(tempos_sessao):.3f}")
print(f"IC 95% Bootstrap da Mediana: [{ic_mediana_inf:.3f}, {ic_mediana_sup:.3f}]")
```
