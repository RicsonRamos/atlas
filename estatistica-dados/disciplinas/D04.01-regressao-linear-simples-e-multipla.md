## Conteúdo Programático
**Fundamentos**: O conceito de correlação linear (Pearson, Spearman) vs. Regressão; predição vs. Inferência estrutural.
**Teoria Estatística**: A equação de regressão linear $Y = \beta_0 + \beta_1 X_1 + \dots + \beta_k X_k + \epsilon$; Mínimos Quadrados Ordinários (OLS) e derivação analítica; Teorema de Gauss-Markov (BLUE - Melhor Estimador Linear Não-Viesado); pressupostos dos resíduos: homocedasticidade (teste de Breusch-Pagan), normalidade (teste Jarque-Bera), independência (teste Durbin-Watson); multicolinearidade (VIF); o trade-off de ajustar preditores adicionais ($R^2$ vs. $R^2$ Ajustado).
**Implementação em Python**: Modelagem com a API de fórmulas do `statsmodels` (`ols("y ~ x1 + x2", data=df).fit()`); análise do relatório `.summary()`; cálculo do VIF com `statsmodels.stats.outliers_influence.variance_inflation_factor`.

## Implementação em Python
```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
import statsmodels.formula.api as smf
from statsmodels.stats.outliers_influence import variance_inflation_factor
from statsmodels.stats.diagnostic import het_breuschpagan

# Criar conjunto de dados sintético
np.random.seed(42)
n = 200
area = np.random.normal(loc=120, scale=30, size=n)
quartos = np.random.randint(1, 5, size=n)
# Preço influenciado por área, quartos e ruído aleatório
preco = 50000 + 1200 * area + 15000 * quartos + np.random.normal(0, 15000, size=n)

df = pd.DataFrame({'preco': preco, 'area': area, 'quartos': quartos})

# 1. Ajustar o Modelo usando Statsmodels com a API de fórmulas (estilo R)
modelo = smf.ols("preco ~ area + quartos", data=df).fit()

# Exibir relatório completo
print(modelo.summary())


# 2. Diagnóstico de Pressupostos
# A. Multicolinearidade (VIF)
# Criar matriz de variáveis explicativas com intercepto
X = sm.add_constant(df[['area', 'quartos']])
vifs = [variance_inflation_factor(X.values, i) for i in range(X.shape[1])]
print("\n--- Fatores de Inflação de Variância (VIF) ---")
print(f"VIF Intercepto: {vifs[0]:.2f}")
print(f"VIF Area: {vifs[1]:.2f} | VIF Quartos: {vifs[2]:.2f} (Valores < 5 indicam ausência de colinearidade)")

# B. Teste de Homocedasticidade de Breusch-Pagan
# H0: Resíduos são homocedásticos (variância constante)
bp_test = het_breuschpagan(modelo.resid, modelo.model.exog)
labels = ['Lagrange Multiplier statistic', 'p-value', 'f-value', 'f p-value']
print("\n--- Teste de Breusch-Pagan para Homocedasticidade ---")
print(f"p-valor do teste BP: {bp_test[1]:.5f}")
```
