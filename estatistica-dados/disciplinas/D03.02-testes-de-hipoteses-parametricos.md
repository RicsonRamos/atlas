## Ementa
Esta disciplina estuda as metodologias de teste de hipóteses paramétricos, que assumem pressupostos sobre a distribuição de probabilidade subjacente dos dados. São abordados os conceitos fundamentais de hipótese nula ($H_0$), hipótese alternativa ($H_1$), nível de significância ($\alpha$), erro do Tipo I (falso positivo) e do Tipo II (falso negativo), poder estatístico ($1-\beta$) e p-valor. No âmbito prático, cobre a execução e validação dos testes t de Student (uma amostra, amostras independentes e amostras pareadas), teste Z para grandes amostras, e a Análise de Variância (ANOVA) de uma via com testes subsequentes de comparação múltipla (post-hoc de Tukey). A validação dos pressupostos de normalidade (Shapiro-Wilk) e homocedasticidade (Levene) é ensinada usando SciPy.

## Objetivos
1. Formular hipóteses nulas e alternativas para problemas de negócios e selecionar o teste estatístico paramétrico adequado.
2. Calcular e interpretar p-valores, contrastando-os com limites críticos e intervalos de confiança.
3. Testar a igualdade de médias entre três ou mais grupos independentes usando ANOVA, validando os pressupostos matemáticos exigidos.

## Conteúdo Programático
**Fundamentos**: O método científico na tomada de decisão baseada em dados; formulação de $H_0$ e $H_1$; testes uni e bicaudais.
**Teoria Estatística**: O significado probabilístico do p-valor ($P(\text{dados observados} \mid H_0 \text{ é verdadeira})$); trade-off entre erro Tipo I e Tipo II; cálculo do poder estatístico; teste t de Student e t de Welch (para variâncias desiguais); pressuposto da homogeneidade de variâncias (homocedasticidade); pressuposto de normalidade; lógica da ANOVA: comparação da variância entre-grupos vs. dentro-grupos (estatística F); o problema das comparações múltiplas e a necessidade de testes post-hoc (Tukey's Honest Significant Difference).
**Implementação em Python**: Testes de normalidade (`stats.shapiro`); testes de homocedasticidade (`stats.levene`); teste t para duas amostras (`stats.ttest_ind` e `stats.ttest_rel`); ANOVA (`stats.f_oneway`) e Tukey HSD (`stats.tukey_hsd` ou `statsmodels.stats.multicomp.pairwise_tukeyhsd`).

## Implementação em Python
```python
import numpy as np
from scipy import stats
from statsmodels.stats.multicomp import pairwise_tukeyhsd

# Criar amostras sintéticas representando conversões de 3 designs de página (A, B e C)
np.random.seed(42)
grupo_A = np.random.normal(loc=12.2, scale=2.1, size=50)
grupo_B = np.random.normal(loc=13.5, scale=1.9, size=50)
grupo_C = np.random.normal(loc=11.8, scale=2.2, size=50)

# 1. Validar pressupostos para os Grupos (A e B) antes do teste t
# Teste de Normalidade (Shapiro-Wilk) -> H0: Os dados são normais
stat_a, p_shapiro_a = stats.shapiro(grupo_A)
stat_b, p_shapiro_b = stats.shapiro(grupo_B)
print(f"Normalidade Grupo A (p-valor): {p_shapiro_a:.4f} | Grupo B: {p_shapiro_b:.4f}")

# Teste de Homocedasticidade (Levene) -> H0: Variâncias são iguais
stat_l, p_levene = stats.levene(grupo_A, grupo_B)
print(f"Homocedasticidade (p-valor): {p_levene:.4f}")

# Como p-valores > 0.05, não rejeitamos normalidade nem igualdade de variâncias.
# Executamos o teste t de duas amostras independentes (bicaudal)
t_stat, p_ttest = stats.ttest_ind(grupo_A, grupo_B, equal_var=True)
print(f"Teste t de Student A vs B -> Estatística t: {t_stat:.3f} | p-valor: {p_ttest:.5f}")


# 2. ANOVA para testar igualdade de médias entre os 3 Grupos (A, B e C)
f_stat, p_anova = stats.f_oneway(grupo_A, grupo_B, grupo_C)
print(f"\nANOVA Unifatorial A vs B vs C -> Estatística F: {f_stat:.3f} | p-valor: {p_anova:.5f}")

if p_anova < 0.05:
    print("Diferença significativa detectada! Executando teste post-hoc de Tukey...")
    # Preparar dados estruturados para statsmodels
    valores = np.concatenate([grupo_A, grupo_B, grupo_C])
    grupos = ['A']*50 + ['B']*50 + ['C']*50
    
    tukey = pairwise_tukeyhsd(endog=valores, groups=grupos, alpha=0.05)
    print(tukey)
```
