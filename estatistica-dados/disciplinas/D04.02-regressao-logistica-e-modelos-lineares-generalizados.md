## Ementa
Esta disciplina aprofunda a modelagem de regressão para variáveis de resposta não-normais e binárias. É apresentada a teoria de Modelos Lineares Generalizados (GLM), detalhando a família exponencial de distribuições e a função de ligação (link function). O foco principal é a Regressão Logística Binária, derivando a transformação Logit, o conceito de Log-Odds (logaritmo da razão de chances) e a interpretação física de Odds Ratio (razão de chances). São discutidos métodos de otimização por Máxima Verossimilhança, métricas de ajuste global (AIC, BIC, Pseudo-$R^2$ de McFadden) e métricas de desempenho de classificação em aprendizado de máquina (Matriz de Confusão, Precisão, Sensibilidade/Recall, F1-Score e Curva ROC-AUC).

## Conteúdo Programático
**Fundamentos**: Limitações do Modelo Linear de Probabilidade (LPM); a família exponencial de distribuições; componentes de um GLM.
**Teoria Estatística**: A função de ligação Logit $\ln(\frac{p}{1-p}) = \beta_0 + \beta_1 X_1$; transformação logística sigmoide; razão de chances (Odds Ratio $e^{\beta_1}$); modelagem de contagens com regressão de Poisson (ligação Log); log-verossimilhança; Pseudo-$R^2$ de McFadden; métricas de classificação em Data Science e o trade-off entre precisão e recall.
**Implementação em Python**: Ajuste de regressão logística usando `statsmodels.api.Logit` ou `smf.glm` com a família `sm.families.Binomial()`; cálculo de Odds Ratios; geração da curva ROC e cálculo da área sob a curva (AUC) com `sklearn.metrics`.

## Implementação em Python
```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
import statsmodels.formula.api as smf
from sklearn.metrics import classification_report, roc_auc_score, roc_curve

# Criar dados de churn sintéticos (0 = Não cancelou, 1 = Cancelou)
np.random.seed(42)
n = 300
tempo_casa = np.random.normal(loc=12, scale=4, size=n)
reclamacoes = np.random.poisson(lam=1.5, size=n)
# Churn influenciado pelo tempo de casa (negativo) e reclamações (positivo)
probabilidade = 1 / (1 + np.exp(-(0.5 - 0.25 * tempo_casa + 0.8 * reclamacoes)))
churn = np.random.binomial(n=1, p=probabilidade)

df = pd.DataFrame({'churn': churn, 'tempo_casa': tempo_casa, 'reclamacoes': reclamacoes})

# 1. Ajustar Regressão Logística Binária via GLM
modelo_logit = smf.glm("churn ~ tempo_casa + reclamacoes", data=df, 
                       family=sm.families.Binomial()).fit()

print(modelo_logit.summary())

# 2. Calcular e interpretar as Razões de Chances (Odds Ratios)
odds_ratios = np.exp(modelo_logit.params)
print("\n--- Razões de Chances (Odds Ratios) ---")
for var, or_val in odds_ratios.items():
    print(f"{var}: {or_val:.4f} (Para cada incremento de 1 unidade, a chance de churn multiplica por {or_val:.2f})")


# 3. Avaliar desempenho de classificação
previsoes_prob = modelo_logit.predict(df[['tempo_casa', 'reclamacoes']])
previsoes_classe = (previsoes_prob >= 0.5).astype(int)

print("\n--- Relatório de Classificação ---")
print(classification_report(df['churn'], previsoes_classe))

auc = roc_auc_score(df['churn'], previsoes_prob)
print(f"Área sob a Curva ROC (AUC): {auc:.4f}")
```
