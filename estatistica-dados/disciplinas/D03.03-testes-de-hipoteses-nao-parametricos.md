## Ementa
Esta disciplina foca nos testes de hipóteses não-paramétricos, recomendados quando os pressupostos de normalidade ou homogeneidade de variâncias exigidos pelos testes paramétricos são violados, ou quando lidamos com dados ordinais e amostras muito pequenas. Os alunos aprenderão a fundamentação lógica baseada em ordenação por postos (ranks) em substituição às médias populacionais. Cobre o teste do Qui-Quadrado (de aderência e de independência), o teste de Mann-Whitney U (equivalente ao teste t de amostras independentes), o teste de Wilcoxon (equivalente ao teste t pareado) e a análise de Kruskal-Wallis (equivalente à ANOVA).

## Objetivos
1. Reconhecer quando as premissas de testes paramétricos são violadas e selecionar o teste não-paramétrico substituto correto.
2. Executar e interpretar testes de independência baseados em tabelas de contingência utilizando o teste do Qui-Quadrado.
3. Aplicar técnicas de ordenação por postos para comparar medianas ou distribuições de grupos independentes e pareados.

## Conteúdo Programático
**Fundamentos**: Limitações dos testes paramétricos; perda de poder estatístico em distribuições não normais; conceito de eficiência assintótica relativa.
**Teoria Estatística**: O teste Qui-Quadrado ($\chi^2$) de Independência (tabelas de contingência, frequências observadas vs. esperadas, correção de Yates); teste de Mann-Whitney U (soma de postos de Wilcoxon para grupos independentes); teste de postos sinalizados de Wilcoxon para amostras pareadas; teste Kruskal-Wallis para comparação múltipla baseada em postos.
**Implementação em Python**: Qui-quadrado (`scipy.stats.chi2_contingency`); Mann-Whitney U (`scipy.stats.mannwhitneyu`); Wilcoxon pareado (`scipy.stats.wilcoxon`); Kruskal-Wallis (`scipy.stats.kruskal`).

## Implementação em Python
```python
import numpy as np
import pandas as pd
from scipy import stats

# 1. Teste de Mann-Whitney U (Substituto do teste t para dados não-normais)
# Exemplo: Número de cliques diários em duas campanhas (altamente assimétricos)
np.random.seed(42)
campanha_A = np.random.poisson(lam=3, size=40)
campanha_B = np.random.poisson(lam=5, size=40)

# Mann-Whitney U -> H0: As distribuições das duas populações são iguais
u_stat, p_mwu = stats.mannwhitneyu(campanha_A, campanha_B, alternative='two-sided')
print(f"Mann-Whitney U -> Estatística U: {u_stat:.2f} | p-valor: {p_mwu:.6f}")


# 2. Teste do Qui-Quadrado de Independência (Variáveis Categóricas)
# Exemplo: Relação entre Gênero e preferência por Tipo de Plano (Básico, Premium)
# Criar uma tabela de contingência
#               Básico   Premium
# Feminino        120       80
# Masculino        90      110
tabela_contingencia = np.array([[120, 80], 
                                [90, 110]])

# Executar Qui-Quadrado de Independência
# H0: Gênero e Preferência por plano são variáveis independentes
chi2_stat, p_chi2, dof, esperados = stats.chi2_contingency(tabela_contingencia, correction=True)

print(f"\nQui-Quadrado -> Estatística Chi2: {chi2_stat:.4f}")
print(f"p-valor: {p_chi2:.5f} | Graus de Liberdade: {dof}")
print("Frequências Esperadas sob H0:\n", esperados)

if p_chi2 < 0.05:
    print("Rejeitamos H0: Há evidências de associação estatisticamente significativa entre Gênero e Plano.")
else:
    print("Não rejeitamos H0: As variáveis são estatisticamente independentes.")
```
