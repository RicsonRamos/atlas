Esta disciplina cobre os fundamentos da estatística descritiva univariada, focando em resumir e descrever as características de um conjunto de dados. São abordadas as medidas de centralidade (média aritmética, ponderada, geométrica, mediana e moda), medidas de variabilidade (amplitude, variância, desvio padrão, desvio médio absoluto e coeficiente de variação), além de medidas de posicionamento relativo e formato de distribuição (percentis, quartis, amplitude interquartil, assimetria e curtose). ## Objetivos

3. Identificar e tratar valores discrepantes (outliers) utilizando o critério do Intervalo Interquartil (IQR).

## Conteúdo Programático
**Fundamentos**: Definição de população e amostra; tipos de variáveis (qualitativas nominais/ordinais, quantitativas discretas/contínuas); o papel do resumo estatístico na ciência de dados.
**Teoria Estatística**: Média matemática vs. Mediana (sensibilidade a valores extremos); variância populacional e amostral (Correção de Bessel); graus de liberdade; interpretação física do desvio padrão; coeficiente de variação para comparação de dispersão adimensional; conceitos de momentos estatísticos (1º momento: média, 2º: variância, 3º: assimetria, 4º: curtose); curtose mesocúrtica, leptocúrtica e platicúrtica.
**Implementação em Python**: Métodos do Pandas (`.describe()`, `.mean()`, `.median()`, `.var()`, `.std()`, `.skew()`, `.kurt()`); cálculo vetorial com NumPy; identificação de outliers com o método IQR.

## Implementação em Python
```python
import numpy as np
import pandas as pd
from scipy import stats

# Criar um conjunto de dados sintético com cauda longa (assimétrico)
np.random.seed(42)
dados = np.random.exponential(scale=10, size=1000)

# Converter para DataFrame Pandas
df = pd.DataFrame(dados, columns=['Valores'])

# 1. Medidas de Tendência Central
media = df['Valores'].mean()
mediana = df['Valores'].median()
moda = df['Valores'].mode()[0]

# 2. Medidas de Dispersão
variancia = df['Valores'].var(ddof=1) # ddof=1 aplica a correção de Bessel
desvio_padrao = df['Valores'].std(ddof=1)
coef_variacao = desvio_padrao / media

# 3. Medidas de Posição e Forma
q1 = df['Valores'].quantile(0.25)
q3 = df['Valores'].quantile(0.75)
iqr = q3 - q1
assimetria = df['Valores'].skew()
curtose = df['Valores'].kurt() # Curtose excessiva do Pandas (Fisher)

print(f"Média: {media:.3f} | Mediana: {mediana:.3f} | Moda: {moda:.3f}")
print(f"Desvio Padrão: {desvio_padrao:.3f} | Coef. Variação: {coef_variacao:.3f}")
print(f"IQR: {iqr:.3f} | Assimetria: {assimetria:.3f} (Positiva) | Curtose: {curtose:.3f}")

# Identificação de Outliers usando IQR
limite_inferior = q1 - 1.5 * iqr
limite_superior = q3 + 1.5 * iqr
outliers = df[(df['Valores'] < limite_inferior) | (df['Valores'] > limite_superior)]
print(f"Quantidade de outliers detectados via IQR: {len(outliers)}")
```
