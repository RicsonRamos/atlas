## Conteúdo Programático
**Fundamentos**: Desenhos experimentais (RCT); o modelo de resultados potenciais de Rubin; a premissa de valor de tratamento estável de unidade (SUTVA).
**Teoria Estatística**: Dimensionamento de amostra para proporções (fórmula de Cohen/fleiss) e médias; MDE (Minimum Detectable Effect); o perigo do "espiamento de dados" (peeking) e interrupção precoce de testes; inflação do erro da família (Family-wise Error Rate - FWER); correção de Bonferroni (conservadora); método de Benjamini-Hochberg para controle de False Discovery Rate (FDR); inferência causal: DAGs causais (Gráficos Direcionados Acíclicos); variáveis de confusão (confounders); pareamento por Propensity Score (PSM); método de Diferença em Diferenças (DiD).
**Implementação em Python**: Dimensionamento amostral com `statsmodels.stats.power`; correção de múltiplos testes com `statsmodels.stats.multitest.multipletests`; regressão de ajuste causal.

## Implementação em Python
```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
import statsmodels.stats.api as sms
from statsmodels.stats.multitest import multipletests

# 1. Dimensionamento Amostral para Teste A/B (Proporções)
# Conversão base do site = 5%. Deseja-se detectar aumento absoluto para 5.5% (MDE = 0.5%)
# Parâmetros: Poder (1-β) = 80%, Significância (α) = 5%
taxa_controle = 0.05
taxa_tratamento = 0.055

# Calcular o Cohen's h (tamanho do efeito para proporções)
effect_size = sms.proportion_effectsize(taxa_controle, taxa_tratamento)

# Dimensionamento
power_analysis = sms.NormalIndPower()
tamanho_amostra = power_analysis.solve_power(effect_size=effect_size, alpha=0.05, 
                                             power=0.80, ratio=1.0, alternative='two-sided')

print(f"Tamanho amostral necessário por variante (A e B): {int(np.ceil(tamanho_amostra))}")


# 2. Correção de Múltiplos Testes (Benjamini-Hochberg para FDR)
# Imagine que testamos 10 novas funcionalidades simultaneamente. Obtivemos 10 p-valores brutos.
p_valores_brutos = [0.003, 0.012, 0.045, 0.071, 0.120, 0.001, 0.033, 0.450, 0.890, 0.021]

# Correção BH (FDR)
rejeitar, p_corrigidos, _, _ = multipletests(p_valores_brutos, alpha=0.05, method='fdr_bh')

df_testes = pd.DataFrame({
    'p-valor Bruto': p_valores_brutos,
    'p-valor Corrigido (FDR BH)': p_corrigidos,
    'Rejeita H0 (Diferença Real?)': rejeitar
})
print("\n--- Correção de Benjamini-Hochberg (α=0.05) ---")
print(df_testes.sort_values(by='p-valor Bruto'))
```
