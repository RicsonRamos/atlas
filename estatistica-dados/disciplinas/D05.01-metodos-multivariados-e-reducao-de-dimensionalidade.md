## Conteúdo Programático
**Fundamentos**: O problema da maldição da dimensionalidade (curse of dimensionality); distância Euclidiana vs. distância de Mahalanobis (corrigindo por escala e correlação).
**Teoria Estatística**: Matriz de covariância $\Sigma$; decomposição em autovalores e autovetores de $\Sigma$ ($Av = \lambda v$); derivação matemática do PCA: o primeiro componente principal direciona-se ao vetor que maximiza a variância dos dados projetados; ortogonalidade dos componentes subsequentes; variância explicada por componente $\lambda_i / \sum \lambda_j$; a importância da padronização dos dados (z-score) antes do PCA.
**Implementação em Python**: Decomposição espectral com `np.linalg.eig`; cálculo de distância de Mahalanobis com `scipy.spatial.distance.mahalanobis`; modelagem de PCA com `sklearn.decomposition.PCA`.

## Implementação em Python
```python
import numpy as np
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA

# 1. Criar dados sintéticos altamente correlacionados (multivariados)
np.random.seed(42)
x1 = np.random.normal(loc=10, scale=2, size=150)
# x2 e x3 são linearmente acoplados com ruído
x2 = 2 * x1 + np.random.normal(0, 1, size=150)
x3 = -0.5 * x1 + np.random.normal(0, 0.5, size=150)

df = pd.DataFrame({'x1': x1, 'x2': x2, 'x3': x3})

# Matriz de covariância original
print("Matriz de Covariância Original:\n", df.cov())


# 2. Padronizar os dados (Essencial para PCA se escalas forem diferentes)
scaler = StandardScaler()
dados_padronizados = scaler.fit_transform(df)

# Ajustar o PCA
pca = PCA(n_components=3)
componentes = pca.fit_transform(dados_padronizados)

# Variância Explicada por cada Componente
var_explicada = pca.explained_variance_ratio_
var_acumulada = np.cumsum(var_explicada)

print("\n--- Resultados do PCA ---")
for i, var in enumerate(var_explicada):
    print(f"Componente {i+1}: {var*100:.2f}% de variância explicada (Acumulado: {var_acumulada[i]*100:.2f}%)")

# Carga dos Componentes (Pesos originais de cada variável nos autovetores)
cargas = pd.DataFrame(pca.components_.T, columns=['PC1', 'PC2', 'PC3'], index=df.columns)
print("\nMatriz de Cargas (Autovetores):\n", cargas)
```
